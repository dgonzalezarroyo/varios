Problema 1: Circuito Serie Simple

Problema 2: Circuito Paralelo

Determinar la resistencia equivalente, corriente total y corriente por cada rama.
?? Haz clic para ver la respuesta
Solución:
• 1/R_eq = 1/8 + 1/12 = 3/24 + 2/24 = 5/24
• R_eq = 24/5 = 4.8Ω
• Corriente total: I_total = 24V/4.8Ω = 5A
• Corriente rama 1: I₁ = 24V/8Ω = 3A
• Corriente rama 2: I₂ = 24V/12Ω = 2A

Problema 3: Circuito Mixto (Serie-Paralelo)

Encontrar la resistencia equivalente total del circuito y la corriente principal.
?? Haz clic para ver la respuesta
Solución:
• Paralelo: 1/R_p = 1/6 + 1/9 = 3/18 + 2/18 = 5/18 → R_p = 18/5 = 3.6Ω
• Serie total: R_total = 3Ω + 3.6Ω + 5Ω = 11.6Ω
• Corriente principal: I = 18V/11.6Ω ≈ 1.55A

Problema 4: Circuito Complejo

Calcular la resistencia equivalente total y determinar la potencia disipada en cada resistencia.
?? Haz clic para ver la respuesta
Solución:
• Paralelo: 1/R_p = 1/4 + 1/8 = 2/8 + 1/8 = 3/8 → R_p = 8/3 ≈ 2.67Ω
• R_total = 2Ω + 2.67Ω + 1Ω = 5.67Ω
• I_total = 30V/5.67Ω ≈ 5.29A
• Potencias: P₁ = I²R₁ = (5.29)²×2 ≈ 56W, etc.
